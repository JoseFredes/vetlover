Para ejecutar el prototipo de aplicación debe tener a su disposición Visual Studio 2019 además de XAMPP en su ultima versión.
1. Debe descomprimir el archivo RAR que se llama "proyectoclinica"
2. Ejecutar el archivo ClinicaVeterinaria.sln, el cual se encuentra dentro de la misma carpeta.
3. Una vez dentro del Visual Studio, se debe dirigir a las Dependencias del programa, darle click derecho, e irse a administrar NuGet. Se requiere de manera obligatoria tener instalado MySql.Data y MySqlConnector
4. Debe descargar el archivo ventas2.sql y levantar el XAMPP desde el administrador, y subir la base de datos recientemente descargada.
5. Los query se encuentan en la clase base de datos
6. Con estos pasos debería poder ejecutar el programa sin problemas y poder visualizar las interfaces programadas.
